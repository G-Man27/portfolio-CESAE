Sistema pictográfico para identificação de medicamentos

O Projeto SPicMed é um projeto escalável e internacionalizável que segue uma classificação de medicamentos já adotada pela OMS e que transforma essa classificação em imagens. Utiliza o modo habitual como as pessoas reconhecem os medicamentos, pela sua análise anatomico-terapêutica (para que serve?). Segue uma lógica de sinais de trânsito para mais fácil memorização.
